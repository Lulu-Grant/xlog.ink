# BRAND

This document defines the current `xlog` logo assets and basic usage rules.

## Primary Asset

- `assets/brand/xlog-mark.svg` — primary logo mark

The current logo combines:

- a rounded document-like silhouette
- a folded corner that hints at publishing and page generation
- a luminous `X` monogram that keeps the mark readable at small sizes

## Recommended Usage

Use the current mark in these contexts:

- site header
- footer brand link
- future favicon or app icon derivations
- social or documentation branding where a compact symbol is needed

For full wordmark usage in the site UI, pair the mark with the `XLOG` text label, as the current header does.

## Color and Contrast

The mark is designed for:

- dark surfaces
- deep navy backgrounds
- light UI surfaces when the SVG is used at normal size

The current gradients intentionally lean toward cyan and blue to match the product's existing accent palette.

## Do

- keep the SVG proportions unchanged
- use the shipped asset directly when possible
- pair it with the `XLOG` wordmark for navigation/header contexts
- keep enough spacing around the mark so the folded corner and `X` stay legible

## Do Not

- stretch or squash the mark
- recolor individual pieces independently without updating the full brand system
- add effects on top of the SVG that reduce legibility
- use the mark as a repeating decorative texture
