快速上手

  • Xlog 提供兩種建立模式：
    ① 連結模式（標題≤30、正文≤500、1–20 條連結：名稱+URL）
    ② 文章模式（Markdown）（標題≤30、正文建議≤5000，支援上色工具）
  • 選擇主題（明/暗）與界面語言，完成人機驗證後提交生成。
  • 系統會分配 https://XXXXXXXXXX.xlog.ink/ 專屬二級網域，可直接分享與使用。

內容規範與限制

允許的成人內容

僅限合法、自願、成年人（18+） 的成人/情色內容；遵守當地法律與平台規範，尊重隱私與肖像權，建議標示 NSFW。

嚴禁內容

  • 詐騙與非法行為（釣魚、詐財、勒索、販售/引導非法服務或工具）。
  • 毒品（宣傳、販售或教唆製作/使用非法藥物）。
  • 仇恨與煽動暴力（基於種族、民族、宗教、性別、性傾向等）。
  • 未成年人保護：任何涉未成年之性內容、性暗示、剝削或虐待；任何形式的兒少性犯罪內容或辯護。
  • 非自願內容：報復性色情、偷拍、強迫、胁迫、藥物控制、迷姦等。
  • 兒童虐待 / 自殘 / 極端暴力之展示或美化。
  • 未經明確同意之真實個體色情深偽（deepfake）。
  • 隱私侵害（人肉、公開敏感個資如醫療/財務資料）。
  • 惡意軟體/黑客（木馬、間諜軟體、惡意腳本或其散布）。

Xlog 得以下線或封鎖違規頁面；使用者對其內容自行負責。

生成規則與配額

  • 連結模式：標題 ≤30；正文 ≤500；1–20 條連結；名稱 ≤30，URL 需以 http/https 開頭。
  • 文章模式：標題 ≤30；正文建議 ≤5000；支援 Markdown 與文字上色（預設 16 色）。
  • 提交前需完成 Cloudflare Turnstile 人機驗證，以降低濫用與批量生成。
  • 生成後頁面為靜態且不可修改/刪除；請在提交前確認內容。
  • 頻率限制：1 分鐘 1 次、24 小時 50 次（預設基於 REMOTE_ADDR；僅在受信代理配置後才讀取轉發 IP）。

文章模式（Markdown）

  • 使用 EasyMDE 編輯器，支援標題、粗體、斜體、清單、程式碼、表格、分隔線、引用等。
  • 工具列包含預覽；已移除展開與全螢幕；新增吸管上色，點擊後出現 16 色預設色塊。
  • 選中文字後點色塊，即會插入 <span style="color:#RRGGBB">...</span> 包裹文字。
  • 若未選中文字，將插入預設占位「彩色文字」。

二級網域與訪問

  • 每個頁面分配一個 10 位字母數字二級網域：https://XXXXXXXXXX.xlog.ink/。
  • 建議於履歷、社交個人檔案、作品集與宣傳頁中使用該連結。
  • 頁面為靜態內容，CDN 加速，通常達到毫秒級響應。

隱私與安全

  • 系統會對正文中的郵件地址做基礎處理以減少垃圾抓取。
  • 請勿上傳或填寫個人敏感資料（身分證/銀行/醫療等）。
  • 若發現疑似違規或侵權內容，請立刻透過頁腳聯繫管道舉報。

SEO 與品牌

  • 生成頁會設置 <title> 與 description；建議簡潔聚焦。
  • 標題建議控制在 30 字內，利於行動端展示與社交分享卡片。
  • 請勿誤導性關鍵字堆疊或濫用品牌詞。

常見問題

生成後能否修改或刪除？

無法修改或刪除。請在提交前確認內容；這可保證長期穩定可訪問。

我可以生成多個頁面嗎？

可以，但單一 IP 有頻率限制（1 分鐘 1 次、24 小時 50 次），且每次提交前都需完成人機驗證。

文章模式的顏色工具為何不是色盤？

為提升兼容性，我們提供 16 色預設色塊。點擊吸管後選擇色塊即可套用。

Markdown 支援哪些格式？

支援標題、粗體、斜體、清單、程式碼、表格、分隔線、引用等；可即時預覽。

頁面會被搜尋引擎收錄嗎？

會。請善用清晰的標題與文字，避免關鍵字堆砌。
